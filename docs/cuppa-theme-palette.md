# Cuppa Chai Website Theme Palette

Source inspected: https://cuppachai.ae/

The live site is built on Wix, so the page contains both brand/theme colors and extra Wix framework colors. For the QR menu, use the curated palette first; keep the full extracted theme list below as reference.

## Custom QR Menu Palette

Palette supplied for the QR menu redesign:

| Role | Hex | Usage |
| --- | --- | --- |
| Deep espresso | `#3F160A` | Strongest dark background, premium footer/hero overlay |
| Burnt mahogany | `#5A1F0E` | Dark panels, offer section gradients |
| Brand brown | `#733B2E` | Primary brand block, selected category states |
| Copper | `#944C1B` | Secondary action, borders, item-price accents |
| Warm copper gold | `#BB8131` | Primary buttons, highlights, active controls |
| Soft tan | `#C4A97F` | Subtle borders, card shadows, dividers |
| Cream | `#F8EDC8` | Page background, light panels, readable card surfaces |
| Sage | `#AAB489` | Fresh-food accents, soft tags |
| Olive | `#616B38` | Secondary buttons, natural accent |
| Deep olive | `#2E3C29` | Dark green action blocks, footer/review accent |
| Forest black | `#172A21` | Deep contrast text/overlay alternative |

Suggested balance:

- Use `#F8EDC8` as the main light background.
- Use `#3F160A`, `#5A1F0E`, and `#172A21` only for high-contrast areas.
- Use `#BB8131` and `#944C1B` for buttons, prices, and important highlights.
- Use `#AAB489`, `#616B38`, and `#2E3C29` for freshness/natural accents.
- Use `#C4A97F` for subtle dividers and borders.
