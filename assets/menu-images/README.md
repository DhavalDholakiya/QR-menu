# Menu Item Images

Upload dish photos in this folder and connect them in `script.js`.

Example:

```js
{
  name: "Poha",
  price: "12",
  description: "Flattened rice tossed with onions, mustard seeds, and mild spices.",
  image: "assets/menu-images/poha.jpg",
}
```

Supported browser-friendly formats: `.jpg`, `.jpeg`, `.png`, `.webp`.

Recommended image size: around `1200 x 900px`, compressed for web.
